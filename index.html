    D4 Research Assistant (Current)

    Researching how Technical Debt affects Open-Source Machine Learning programs. Using text-mining approaches and the mining of AST nodes from reputable AI programs to discover findings regarding ML antipatterns and their resolutions.

    CS4620: Drone Face Tracker

    The goal of the project was to create an "intelligent agent". Using Sklearn, OpenCV, and PyParrot the group was able to use a drone's onboard camera to identify a face, then classify a set of movements to reposition the face in the center of the drone's frame in real-time.
